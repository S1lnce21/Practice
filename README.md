Установка всех зависимостей
```bash
npm install
```
Установка дополнительных пакетов (бэкенд)
```bash
npm install express cors pg dotenv
```
Установка dev-зависимостей
```bash
npm install -D nodemon concurrently
```

Запускаем проект 
  Терминал №1
```
npm run dev
```
  Терминал №2
```
npm start
```
